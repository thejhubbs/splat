A responsive, simple, single-page website template

A project to design & develop a completely responsive & customizeable layout from scratch

No Bootstrap, No JQueryUI, no pre-written code or libraries, no design-source files. A display of knowledge of designing & building components from scratch.
Powerful. Sleek. Customizeable.

Making This Template Different

Customizeable Design

Drastically & quickly change the appearance of the template by changing fonts, styles, and other things like border radius all from an easy file.

See Full List

Colorize

All colors you see now are generated from one "main" color. Choose yours and instantly theme all other colors to match. Of course, these generations are easily overrideable for the designers out there.

See Examples

Reusable HTML

Standard 12 column (boxes) responsive design, including classes to hide and display at breakpoints. Comes with a plethora of HTML objects to use as you see fit.
